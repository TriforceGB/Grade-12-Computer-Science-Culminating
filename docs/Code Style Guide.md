## Naming 
- Classes = PascalCase
- Functions = camelCase (also should be a verb)
- Variables = camelCase
- Constants = UPPER_CASE

## Comment
- Each Class and Function should have a docstring that display what the function does, what parameters it takes, and what it returns.
- Variables should have comments if they are not self-explanatory.
	- Generally Variables should be understandable without a comment.
