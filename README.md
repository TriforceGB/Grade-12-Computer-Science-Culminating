# Filal Tracking Service (FTS)
The FTS is a local media tracking service for Movies, Shows and Anime. This was our final project from Grade 12 Computer Science.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)

## Installation
If you wish to built the Jar for the FTS you can however, this project was built to run directly from the source code. There may be bugs which we haven't found while compiling.

Additionally, for the FTS to work, you going to need API Key for TheTVDB and AniDB. You can obtain these keys by signing up for an account on their respective websites and enter them into the Program.

## Usage
By default, the FTS will start with an empty database and no API keys. You will need to enter your API keys into the Program before it can work.

The API keys can be enter with the default admin user, (username: admin, password: admin) and from there you can start adding shows to your local DB.

To do this enter the Search tab and enter the name of the show you wish to add. Once you have entered the name, click the Search button and it will start to display shows that match your search query.

Click on a show to add it to your local DB and it will sort the infromation and save the image to your local storage.
